# Скрипт для получения курсов валют

Этот скрипт загружает курсы валют к доллару США с помощью API, сохраняет данные в JSON-файл и создаёт таблицу в форматах XLSX и CSV.

## Что делает скрипт
1. Отправляет запрос к ExchangeRate-API, чтобы получить курсы валют к доллару США.
2. Сохраняет полученные данные в файл `backup.json`.
3. Создаёт таблицу с двумя столбцами:
   - `Currency`: код валюты.
   - `Rate_to_USD`: курс валюты к доллару США.
4. Сохраняет таблицу в файлы `rates.xlsx` (для Excel) и `rates.csv` (текстовый формат).

## Требования
- Установленный Python.
- Установленные библиотеки: `requests`, `pandas`, `openpyxl`.

## Как запустить
1. Сохраните скрипт в файл с названием, например, `exchange_rates.py`.
2. Откройте командную строку или терминал и перейдите в папку, где лежит файл скрипта:
   ```
   cd путь/к/вашей/папке
   ```

3. Запустите скрипт:
   ```
   python exchange_rates.py
   ``````

## Результат
После запуска в той же папке появятся или перезапишутся три файла:
- `backup.json`
- `rates.xlsx`
- `rates.csv`
